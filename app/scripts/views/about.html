Qu’est ce que une API ? C’est une façade par laquelle un logiciel offre des services à d'autres logiciels. Elle est offerte par une bibliothèque logicielle ou un service web, le plus souvent accompagnée d'une description qui spécifie comment des programmes consommateurs peuvent se servir des fonctionnalités du programme fournisseur .

Etyssa met à disposition une API REST, soit un ensemble structuré d’URLs permettant de directement consulter et éditer vos données de plate-formes. Il y a au moins deux cas d’utilisations de ces données :

  • Etyssa consomme ces données et les intègre aux plate-formes habituelles
  • Les collectivités du réseau, Etyssa ou des développeurs extérieurs réalisent des applications qui viennent s’ajouter aux plate-formes classiques

L’API va vous offrir une grande liberté dans la façon dont vous souhaitez traiter et afficher vos données. Cela laisse la place à une dimension expérimentale importante et vous permet de réaliser des projets non prévus à priori par les modules classiques des applications (sous réserve d’étude préalable par nos soins).

Voici une liste (non exhaustive) des données qui peuvent être consultées :

  • Le réseau GET http://api.etyssa.com/rest/services
  • Tous les utilisateurs d’un service GET http://api.etyssa.com/rest/services/{serviceId}/users
  • Un d’utilisateur d’un service GET http://api.etyssa.com/rest/services/{serviceId}/users/{userId}
  • Toutes les annonces d’un service GET http://api.etyssa.com/rest/services/{serviceId}/entries
  • Une annonce d’un service GET http://api.etyssa.com/rest/services/{serviceId}/entries/{entryId}
  • Tous les univers d'un service GET “http://api.etyssa.com/rest/services/{serviceId}/categories
  • Informations sur un service GET http://api.etyssa.com/rest/services/{serviceId}”
  • Ajouter une annonce POST http://api.etyssa.com/rest/services/{serviceId}/entries” (avec authentification)
  • Tous les univers d’un service GET http://api.etyssa.com/rest/services/{serviceId}/categories
  • etc.

Par rapport à l’ouverture de l’application d’Etyssa en license Libre et / ou Open Source, la consommation de l’API vous laisse plus de libertés dans les technologies utilisées (vous pouvez utiliser celle de votre choix, ce qui demande peu d’investissements et vous permet de manipuler l’API avec les outils de votre choix). Vous n’avez pas de serveur ni de base de données à gérer.

Pour Etyssa, le développement de l’API ouvre la voie à l’utilisation de technologies beaucoup plus modernes, basées sur des outils Open Source, tout en se reposant sur les socles éprouvés de notre moteur de profiling et une base de données bien garnie. Les atouts sont la rapidité, flexibilité, réactivité (JS client), l' angularité.
